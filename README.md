## ✨ 功能一览

- 🎁 **一键预设场景**：日常全套、快速日常、挂机辅助、基质养成……四套常用场景预设一键切换，省去繁琐配置！✨
- 🎮 **实时开荒辅助**：开荒小帮手持续在线！实时识别画面，按需补上关键操作，让你专心打就完事了 ✨
    - **自动剧情**：一键跳过对话、智能选分支、快速过剧情，爽就完事了！（当然你想慢慢看也行~）
    - **交互处理**：自动点指引光标（老普之手）、翻说明书、点点点继续、关档案语音，省心不手忙脚乱！
    - **辅助战斗**：自动普攻、闪避、放技能、开大招、打连携，还能锁定目标集火，战斗从此变得轻松写意✨
    - **自动拾取**：看到闪光就捡，绝不放过任何掉落物和采集点，资源收集小能手就是你！
    - **便捷操作**：秒速传送确认、自动滑滑索，赶路效率 MAX！
    - **顺手解拼图**：开荒路上遇到拼图？交给 AI 秒解（也可以直接开「🧩解拼图」循环）😎
    - **拍一拍**：驮兽也安排上，和自动拾取绝配（建议拍完迅速换🐮）🐾
- 🧩 **拼图大师**：AI 火眼金睛，谜题秒解！单次/循环/演示三模式（学习一下大佬怎么解题 😎）。
- 💎 **基质筛选**：智能识别基质词条，按武器稀有度、基质类型和扩展规则自动锁定/废弃，再也不用一个个看属性了！✨
- 🎱 **基质刷取**：自动挑战重度淤积点，支持循环刷取、自动领奖，还能战后直接衔接基质筛选，一条龙轻松刷基质！
- 🌿 **环境监测**：自动完成环境监测任务，打怪跑图全包揽！
- ⚔️ **协议空间**：自动消耗理智刷取协议空间，多种奖励路线随心选，清体力绝佳帮手！
- 🌾 **生态农场**：自动收菜、种植、浇水，还支持任务前后顺手整理背包，农场主省心不少！
- 🧺 **自动采集**：野外资源路线自动巡回采集，叶子菌子矿石能捡的尽量都给你捡回来！
- 🎁 **赠送干员礼物**：自动前往联络台，给干员送礼并顺手领取回礼，好感度慢慢拉满！
- 🔫 **武库升级 & 生产制造**：武器升级、装备制造、简易制作、批量简易制作自动化，解放双手，轻松提升战力和产能！💪
- 💰 **自动倒卖**：自动囤货、售卖弹性需求物资双线配合，盯利润、控调度券，倒卖党狂喜！💸
- 🛒 **售卖产品**：各据点产品自动兑换调度券，点点点就交给它吧！
- 🚚 **送货任务**：支持接取转交、仅接取、仅装箱三种模式，按地区开关更灵活！
- 🌆 **委托抢单**：委托列表自动抢单，还能按需传送到仓储节点继续后续操作。
- 💳 **信用点购物**：自动购买信用商店物品，支持优先级、折扣、保留阈值和信用点获取联动，挂机购物两不误！
- 🐌 **库存转移**：帝江号仓库跨区搬家，电池/矿/砂叶等物资一键挪窝，囤囤鼠狂喜！
- 📐 **蓝图搬运工**：建筑党福音！批量导入蓝图，智能解析分享码/杂乱文本，工业流水线说建就建，效率拉满！
- 🎁 **基建任务**：自动领取基建产物、补货，线索收取/放置/赠送一条龙；线索交流也能安排上！
- 🤝 **拜访好友**：自动拜访好友，顺手完成生产助力和情报交流，社交日常轻松清掉！
- 👥 **批量加好友**：一键批量发送好友请求，支持指定 UID 或直接加陌生人，再也不用一个个手打啦！
- 💬 **Baker 嘴替**：自动清掉会话消息，聊天红点和提示一并收拾干净！
- 📅 **日常自动化**：日常奖励（任务/通行证/邮件/暂存区/活动）、模拟空间奖励、百科红点、简制手册领取……一键清空！✅
- 💊 **自动吃药**：应急理智加强剂一键开吃，省一步是一步~💊
- 💻 **多端适配**：支持 PC 端 (Win32 默认/后台/前台) 与安卓端 (ADB)，无论你在哪个平台，我们都能陪你摸鱼~ 🐟
